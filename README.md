# components-callout

## Install

```bash
npm install --save components-callout
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'components-callout'
import 'components-callout/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
